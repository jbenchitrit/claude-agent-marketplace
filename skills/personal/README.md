# Personal Skills

Personal productivity workflow skills. These are individual-specific and may reference personal preferences, tools, or processes.

**Visibility**: `personal` — not intended for public sharing.

## Adding a personal skill

```
skills/personal/
└── my-personal-skill/
    ├── SKILL.md
    └── references/
        └── context.md
```

Register the skill in `catalog.json` with `"visibility": "personal"`.
