# Memory Templates

CLAUDE.md templates for different project and context types.

Copy the relevant template to your project root or `~/.claude/CLAUDE.md` to prime Claude with the right context for a session.

## Template naming convention

```
memory/
├── work-project.md      # Template for work/org projects
├── personal-project.md  # Template for personal projects
└── open-source.md       # Template for open-source contributions
```

## Usage

```bash
cp memory/work-project.md /path/to/your/project/CLAUDE.md
```
